# Hero & Layout Configuration

## Hero Section

- **Dimensions**:
  - Mobile: `60vh` height
  - Tablet: `80vh` height
  - Desktop: `100vh` (Full Screen) height
- **Width**: 100% Full Width (No max-width constraints)
- **Content**: 3 Image Slides (Image Only, No Text Overlay)

## Navbar

- **Type**: Sticky (Stays at top)
- **Responsiveness**:
  - Mobile: Hamburger Menu + Compact Icons
  - Desktop: Full Mega Menu + Search Bar
